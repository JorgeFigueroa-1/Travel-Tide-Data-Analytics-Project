## Overview
**Travel Tide** is a data analytics project focused on understanding booking behaviors and customer segmentation in the travel industry. The goal was to leverage data-driven insights to enhance personalization and optimize perk targeting strategies.

## Objectives
- Analyze large-scale booking data to identify behavior patterns and travel preferences.  
- Engineer features and build a **weighted scoring system** for user segmentation.  
- Detect high-value user clusters such as **frequent flyers**, **last-minute bookers**, and **luxury travelers**.  
- Visualize engagement and booking trends to support strategic decisions.  
- Deliver actionable recommendations to improve data integrity and perk relevance.

## Data
- Thousands of booking records including customer demographics, travel details, and perk usage.  
- Cleaned and preprocessed data to ensure consistency and reliability.  

## Methodology
1. **Data Preprocessing:** Cleaning, handling missing values, and feature normalization.  
2. **Feature Engineering:** Constructed variables capturing booking frequency, timing, and spend level.  
3. **Scoring Model:** Designed a weighted system to evaluate user value and engagement.  
4. **Segmentation:** Applied clustering techniques to identify distinct traveler types.  
5. **Visualization:** Built analytical dashboards with **Seaborn** and **Matplotlib** for stakeholder reporting.

## Tools & Technologies
- **Python** (Pandas, NumPy, Scikit-learn)  
- **Seaborn** & **Matplotlib** for visualization  
- **Jupyter Notebook** for analysis and documentation  

## Key Insights
- Identified clear behavioral clusters driving personalization opportunities.  
- Provided evidence-based recommendations to enhance perk targeting.  
- Improved understanding of customer engagement cycles and booking intent.  

## Results
- Enhanced **data integrity** and **perk relevance** through targeted improvements.  
- Supported **data-driven decision-making** for marketing and retention strategies.
